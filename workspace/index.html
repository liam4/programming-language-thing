View the console!

You can write your own code in PLT at workspace.js, assuming you're running this from a local machine.

Learn more about Programming Language Thing here.

Do note when you run from the browser you get separate built in functions from when you run using node.

In order to make your life easier, I've made a quick script to automatically build everything. Just use `./workspace_dev` inside of `workspace` while working on the code and you'll be set.

(The script requires browserify and nodemon to be installed.)
